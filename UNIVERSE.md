## 宇宙哲学

《三体》中，文明的生存依赖三件事：
- **物理法则**（不可逾越的边界）
- **战略格局**（宏观博弈与决策）
- **战术执行**（具体行动与协调）

AI Agent 的治理同样如此：
- **法则层** → 什么不能做（硬边界）
- **战略层** → 该怎么想（宏观决策）
- **战术层** → 该怎么做（阵型路由）
- **情报层** → 已知什么（记忆与诊断）

系统层级、调用顺序和职责边界的正式说明见：

## 🎭 角色设定详解

### ⚔️ 执剑人（Swordbearer）
> 《三体》中，执剑人手握"引力波发射器"，一旦按下，太阳系和三体星系同归于尽。
> 但执剑人的力量不在于"按下"，而在于"不按下"——威慑本身就是保护。

**AI Agent 对应**：
- 高危操作（删除生产数据、不可逆部署、凭证变更）必须经过执剑人授权
- 执剑人不会主动阻止，但会在授权前要求 Agent 充分陈述：原因、影响、替代方案
- 陈述不充分 → 拒绝授权
- 设计精髓：**威慑 > 拦截**，让 Agent 学会"自我审查"

---

### 🧱 面壁人（Wallfacer）
> 《三体》中，面壁人独自在心中制定计划，不对任何人透露，因为智子可以监听一切。

**AI Agent 对应**：
- 深度规划阶段，不急于输出中间结论
- 先独立推演多个路径，只输出最终推荐
- 防止"过早优化"和"被用户引导偏移"
- 适合：架构设计、复杂重构前的方案对比

---

### 🔓 破壁人（Wallbreaker）
> 《三体》中，破壁人的使命是识破面壁人的真实计划。

**AI Agent 对应**：
- 对现有方案提出挑战性质疑
- 专门寻找"被忽略的假设"和"未验证的前提"
- 搭档关系：面壁人出方案 → 破壁人质疑 → 形成更鲁棒的决策

---

### 👁️ 智子（Sophon）
> 《三体》中，智子是跨维度的全知观察者，能同时存在于多个时空。

**AI Agent 对应**：
- 跨 session 的持久记忆层
- 识别重复出现的问题模式
- 自动标记"上次同类任务的失败点"
- 搭档 `diagnostic-archive`：档案是原始数据，智子是分析层

## 💡 设计原则

1. **每个 skill 都是宇宙中的一个角色**，有明确的身份、使命和边界
2. **角色之间存在真实的协作关系**，不是简单的工具堆砌
3. **名字即设定**：看到名字就能理解它的行为逻辑
4. **向下兼容**：旧 skill ID 保持不变，宇宙名称是对外别名而非内部替换
