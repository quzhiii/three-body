# Downstream Skill Mapping — 阵型到下游 skill 映射

本文件是 `agent-work-environment-v3` 的权威下游映射表。

如果其他文档与本文件冲突，以本文件为准。

---

## 权威映射表

| 阵型 | 默认推荐 | 可替代类型 | 交接目标 |
|---|---|---|---|
| 观机阵（研究模式） | `brainstorming` / `review` | 研究、调研、分析类 skill | 先理解问题，再给出证据与结论 |
| 破局阵（实现模式） | `Code` / `test-runner` | 编码、测试、修复类 skill | 动手修改并完成最小验证 |
| 明鉴阵（验证模式） | `review` / `qa` | 验证、审查、测试类 skill | 找发现、给证据、做风险判断 |
| 立言阵（写作模式） | `doc` | 文档、方案、总结类 skill | 输出结构化文档与行动项 |
| 行令阵（运维模式） | `careful` / `guard` | 环境、部署、命令治理类 skill | 在保护边界下执行命令或环境变更 |

---

## 回退规则

- 推荐 skill 不存在时，退化为对应类型的 skill，不虚构依赖
- 用户已显式指定 skill 且与阵型一致时，优先尊重用户指定
- 用户指定的 skill 与阵型不一致时，先说明偏差，再决定是否切阵
